Resultados finales

Los resultados se guardarán en bases de SQLite, cuyos nombres están dados por el archivo de configuración y se ubian en data/db/. Existe una base para los datos de las etapas, viajes y otros que se van actualizando a medida que se ingresan nuevos datos, y otra base de insumos para información que no se actualiza con tanta periodicidad (como por ejemplo los recorridos, parads, matrices de las distancias entre puntos fijos de una ciudad).
